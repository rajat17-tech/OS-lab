Learn Operating Systems Through Interactive Simulations

This collection of simulators helps students and enthusiasts understand fundamental operating system concepts through hands-on visualization. Each tool demonstrates a different aspect of OS functionality with real-time feedback and visual representations.

CPU Scheduling Simulator

Visualize and compare First-Come-First-Served (FCFS) and Shortest-Job-First (SJF) CPU scheduling algorithms with interactive Gantt charts.

Key Features:

  • FCFS and SJF algorithms
  • Interactive Gantt charts
  • Waiting and turnaround time calculations
  • Process table visualization
