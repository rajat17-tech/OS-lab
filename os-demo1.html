🔄 Booting Process

The booting process is the sequence of events that occurs from the moment you turn on your computer until the operating system is fully loaded.

BIOS/UEFI
Initializes hardware and performs self-test
POST
Power-On Self-Test checks system components
Bootloader
Loads the operating system kernel into memory
Kernel Load
OS kernel takes control of the system
Init
Initializes system services and user interface

🖥️ Types of Operating Systems

Batch OS

Executes jobs in batches without user interaction. Ideal for repetitive tasks.

Time-Sharing OS

Allows multiple users to share system resources simultaneously.

Distributed OS

Manages a group of independent computers as a single system.

Network OS

Provides services over a network and manages network resources.

Real-Time OS

Processes tasks with precise timing constraints for critical applications.

Mobile OS

Optimized for handheld devices with touch interfaces and mobility.
